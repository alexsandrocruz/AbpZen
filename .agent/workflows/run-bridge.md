---
description: how to run the ZenCode Bridge server
---
1. Navigate to the zencode-generator directory:
   ```bash
   cd /Users/alexsandrocruz/Documents/dev/AbpZen/zencode-generator
   ```
2. Start the bridge server using Node.js:
   ```bash
   node bridge.js
   ```
3. The bridge will be available at `http://localhost:3001` and will handle file operations for the generator.
   
// turbo
4. Run the generator frontend (if not already running):
   ```bash
   npm run dev
   ```

# Text Template Management

This module is used to store and edit template contents for the text templating system of the ABP Framework.

## Permissions

After the module installation, the following permissions are added to the system:

- **TextTemplateManagement**
  - **TextTemplateManagement.TextTemplates**
    - **TextTemplateManagement.TextTemplates.EditContents**

You may need to give these permissions to the roles that you want to allow to access the Text Template Management UI.

## Documentation

For more information, see the [module documentation](https://abp.io/docs/latest/modules/text-template-management).

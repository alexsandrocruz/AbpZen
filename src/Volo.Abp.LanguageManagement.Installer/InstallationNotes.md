# Language Management

This module is used to manage application languages and localization texts.

## Permissions

After the module installation, the following permissions are added to the system:

- **LanguageManagement**
  - **LanguageManagement.LanguageTexts**
    - **LanguageManagement.LanguageTexts.Edit**
  - **LanguageManagement.Languages**
    - **LanguageManagement.Languages.Edit**
    - **LanguageManagement.Languages.Create**
    - **LanguageManagement.Languages.Delete**
    - **LanguageManagement.Languages.ChangeDefault**

You may need to give these permissions to the roles that you want to allow to access the Language Management UI.

## Documentation

For more information, see the [module documentation](https://abp.io/docs/latest/modules/language-management).
